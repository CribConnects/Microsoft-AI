✅ Foundry → Copilot Studio

Connect Foundry agents to Studio via A2A protocol. One-click publishing to Microsoft 365. Fully supported.

✕ Copilot Studio → Foundry

No direct export path. Assets must be manually rebuilt for Foundry. Start with Foundry if enterprise migration is expected.

Recommended approach: adopt a hybrid strategy — empower business users with Copilot Studio for rapid prototyping, while reserving Microsoft Foundry for complex, mission-critical agents requiring custom models or enterprise backend integration.
